There are 4 fields when you add a chat room.

IP

This field (by default) is chat.etcg.pw. Unless you are using a custom server, do not alter this field.

Room

The name of the room to add (where messages will be sent ro and received from). This has to be decided in advance with whomever you chat with. Capitalization matters.

Nickname

The nickname to use when you connect to the chat room. All messages you send will be signed with this name.

Password

The password to use to decrypt messages. The password has to be decided in advance between you and whomever you are chatting with. If your passwords do not match, then you will not be able to see each other's messages.
