There are 3 parts to each room on the main page

Color

The color circle to the left of each room helps identify which room is which, using color coding. The circle starts off as white (the default) whenever you add a new room. To change the color to your liking, tap on the circle to bring up a selection dialog.

Room Name

This field is simply the name of the chat room.

IP

This field signifies which IP the chat room belongs to.

Tap on the room you just added to open it!
